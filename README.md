# Number Guessing Game 🎯

A simple console-based Number Guessing Game built with C#.

## About

This project generates a random number and asks the player to guess it. The game provides hints whether the guess is too high or too low until the correct number is found.

This project was created to practice:

- Variables
- Loops
- Conditional Statements
- User Input
- Random Number Generation
- Basic C# Programming

## Features

- Random number generation
- Unlimited guessing attempts
- High/Low hints
- Win message when the correct number is guessed

## Technologies Used

- C#
- .NET

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/number-guessing-game.git
```

2. Open the project folder.

3. Run the application:

```bash
dotnet run
```

## Example

```text
Guess the number between 0 to 101: 93
Your number is greater than the actual number!

Guess the number between 0 to 101: 91

Yahoo! You get it.
The number is 91
Do you want to play again?(Y/N): yes
```
